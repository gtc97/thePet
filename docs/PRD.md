# thePet 产品需求文档（PRD）

## 一、产品概述

### 1.1 产品定位
宠物档案记录与展示为核心，上门喂养、在线沟通为附加增值功能，整体轻量化、无冗余功能。

### 1.2 目标用户
- **宠物主**：有宠物日常记录、展示分享需求的养宠人群
- **上门师傅**：提供上门喂养、宠物照看服务的从业者

### 1.3 产品原则
- 严格区分首期开发与待定功能，直播功能仅预留
- 用户隐私数据（微信、手机号、地址）默认私密，地址脱敏展示
- 普通用户与管理端账号完全隔离，数据不互通
- 核心数据（宠物档案、日记、相册）防丢失、防误删
- 禁止复杂社交、排行榜、内容推荐、会员体系

## 二、用户端功能模块

### 2.1 注册登录与双身份体系（首期 ✅）

**功能要点：**
- 手机号验证码注册、登录、密码找回
- 所有用户默认宠物主身份
- 上门师傅资质申请：上传资料 → 后台审核 → 身份切换按钮出现
- 未审核用户全局隐藏接单/服务入口
- 切换身份后菜单、功能入口自动适配

**权限隔离：**
- 宠物主：宠物管理、下单、评价、收藏
- 上门师傅：接单大厅、服务执行、路线规划、收益查看

### 2.2 多宠物档案管理（核心 ✅）

**字段设计：**
- 昵称、品种、年龄、性别、饮食习惯、禁忌事项、头像、自定义备注

**功能：**
- 单账号无限数量宠物，独立档案数据隔离
- 卡片式列表布局（头像、昵称、品种）
- 新增、编辑、删除、查看详情
- 删除宠物同步级联清空相册、日记、服务记录
- 创建上门订单支持单选/多选宠物绑定

### 2.3 宠物展示、分享与隐私设置（核心 ✅）

**宠物详情页：**
- 完整档案、照料备注、历史服务记录、用户评价数据

**分享功能：**
- 宠物卡片分享至微信好友、朋友圈
- 分享弹窗自定义勾选：是否展示相册、成长日记、服务记录

**隐私双模式：**
- 公开：外部用户可查看宠物资料、相册、日记
- 私密：仅自己可见，无法被分享、无法被外部查看

**全局隐私设置：**
- 微信、手机号、地址默认私密
- 仅支持「仅自己可见」「订单关联用户可见」，无公开选项

### 2.4 宠物相册（核心 ✅）

- 每只宠物独立专属相册，数据归属单一宠物
- 支持单张/批量上传图片、短视频
- 前端自动压缩资源
- 图片预览、单素材删除、保存至本地
- 按创建时间倒序展示，最新内容置顶
- 服务对比图/成果图自动归类至对应宠物相册
- 区分「用户日常素材」和「服务记录素材」

### 2.5 宠物成长日记（核心 ✅）

- 单宠物独立日记模块
- 图文编辑，自定义文字配图
- 自动生成系统时间戳，用户不可修改创建时间
- 可编辑内容、删除单条日记
- 重点日记置顶功能（过敏、疾病、喂养禁忌、特殊习性）
- 日记内容支撑对外分享展示（用户自主开关）

### 2.6 宠物数据统计与档案封存（核心 ✅）

**轻量化数据看板：**
- 饲养天数、日记总数、相册总数、上门服务次数

**档案封存：**
- 永久保留相册、日记、服务记录
- 自动关闭上门订单下单入口
- 档案不可删除、不可编辑，仅展示查看

### 2.7 上门喂养订单（附加 ✅）

**极简订单流程：**
- 服务项：喂食、换水、铲屎、简单环境清理
- 自定义上门日期、时间段
- 填写居家注意事项、宠物喂养要求、开门方式备注

**师傅操作：**
- 接单后可查看脱敏地址、服务要求、宠物禁忌、上门时间
- 支持接单、拒单（必须选择原因，同步推送通知）

**订单状态流转：**
```
待接单 → 已接单 → 服务中 → 已完成 → 已评价
  ↓         ↓
已取消    已取消
```

### 2.8 押金风控体系（附加 ✅）

- 下单必须支付平台统一押金，支付成功订单方可生效
- 服务完成、用户验收无误后，押金自动原路退回
- 纠纷场景后台可人工冻结押金、执行赔付退款
- 用户端押金中心展示全部流水明细

### 2.9 服务成果与售后申诉（附加 ✅）

**服务执行：**
- 师傅抵达地址后点击「开始服务」
- 结束服务点击「结束服务」
- 服务时长系统自动统计，时间不可篡改
- 支持上传宠物状态图、清洁对比图、文字总结
- 可视化服务成果同步存入宠物相册

**售后申诉：**
- 订单完成7天内可提交图文售后申诉
- 后台人工仲裁

### 2.10 上门直播（附加 ❌ 首期不开发）

- 仅预留前端入口、后端接口、后台管控页面
- 第二版本迭代实现推流、播放逻辑

### 2.11 地图定位与路线规划（附加 ✅）

- 接入高德地图SDK，支持地址定位
- 师傅接单后支持一键规划步行/驾车路线
- 用户地址全局脱敏，仅订单关联师傅可查看模糊地址

### 2.12 双向评价体系（附加 ✅）

- 订单完成后双向评价
- 宠主给师傅1-5星评分、图文评价
- 师傅评价用户居家环境、配合度
- 师傅个人主页展示综合评分、服务总量、好评率

### 2.13 黑名单体系（附加 ✅）

- 宠物主可拉黑师傅、师傅可拉黑用户
- 拉黑后双方无法产生订单、无法私聊
- 黑名单列表查看、解除拉黑
- 双向拉黑状态提示，规避无效操作

### 2.14 轻量化私聊（附加 ✅）

**限制条件：**
- 无陌生人社交
- 仅订单有效期 + 订单完成7天内的供需双方可私聊
- 聊天仅支持文字、图片消息

**风控：**
- 系统自带敏感词拦截
- 自动拦截索要微信、手机号、引流、广告
- 用户可一键关闭全部私信通道

### 2.15 通用基础功能（首期 ✅）

1. **消息推送**：订单状态、资质审核、售后、评价、系统公告全局推送
2. **宠物收藏**：用户可收藏任意公开宠物主页
3. **意见反馈+客服**：整合统一入口，支持图文匿名反馈、投诉、建议

## 三、管理端后台功能模块

### 3.1 后台账号体系 ✅
- 独立账号密码登录，与用户端完全隔离
- 超级管理员单角色，拥有全部权限
- 支持修改密码、登录日志记录

### 3.2 用户管理与资质审核 ✅
- 全量用户列表、身份、注册信息、账号状态
- 师傅资质审核：通过/驳回，驳回填写备注，自动推送通知
- 用户账号禁用、解封，记录风控日志

### 3.3 宠物内容风控 ✅
- 审核全平台宠物档案、相册图片、成长日记、分享卡片
- 下架违规图片、删除违规日记、屏蔽违规分享
- 批量管控违规账号宠物内容

### 3.4 订单与售后仲裁 ✅
- 多条件筛选查询全部订单
- 查看完整订单详情、服务时长、服务成果
- 处理用户售后申诉，人工介入纠纷仲裁

### 3.5 押金资金管理 ✅
- 自定义配置平台统一押金金额
- 全部押金流水查看
- 纠纷人工冻结、赔付、退款操作

### 3.6 评价与聊天风控 ✅
- 管理全平台用户评价，删除违规、恶意评论
- 查看用户私聊记录，清理违规内容
- 封禁导流、骚扰违规账号

### 3.7 直播监控 ❌
- 仅预留页面与接口，跟随用户端直播功能二期上线

### 3.8 平台配置与数据概览 ✅
- 配置服务项目、平台规则
- 发布、编辑、下架平台公告
- 首页轻量化数据看板（用户、宠物、订单、待处理工单）

### 3.9 用户反馈管理 ✅
- 汇总所有用户反馈
- 标记处理状态、备注优化方案
- 统计高频问题，留存迭代台账

## 四、全局约束

### 必做项
1. 宠物档案、相册、日记、展示分享、隐私管控、数据统计、档案封存
2. 极简订单、押金、服务成果、售后、评价、黑名单、限时私聊

### 预留项（首期不做）
- 上门直播全功能
- 后台直播监控

### 禁止项
- 复杂社交功能
- 排行榜、内容推荐
- 会员体系
- 复杂营销功能
