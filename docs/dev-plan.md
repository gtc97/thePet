## 技术债务与注意事项

1. **短信服务** — 当前为模拟实现，需接入阿里云/腾讯云短信SDK
2. **微信支付** — 押金支付需配置微信支付商户号
3. **微信登录** — 当前使用模拟openid，需配置小程序AppID和Secret
4. **地图SDK** — 高德地图API Key需配置
5. **文件存储** — 当前本地存储，生产环境建议迁移至OSS/CDN
6. **WebSocket** — Socket.IO需在生产环境配置nginx反向代理
7. **Redis** — 当前优雅降级（无Redis仍可运行），但生产环境必须配置
8. **UniApp X** — 需HBuilderX最新版编译，部分API可能有兼容性差异
